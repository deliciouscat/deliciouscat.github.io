Abstract Factory PatternмқҖ кҙҖл Ё мһҲлҠ” к°қмІҙл“Өмқҳ л¬¶мқҢ(м ңн’Ҳкө°)мқ„ мғқм„ұн•ҳлҠ” мқён„°нҺҳмқҙмҠӨлҘј м ңкіөн•ҳлҗҳ, кө¬мІҙм Ғмқё нҒҙлһҳмҠӨлҠ” нҒҙлқјмқҙм–ёнҠёк°Җ м•Ңм§Җ лӘ»н•ҳкІҢ н•ҳлҠ” мғқм„ұ л””мһҗмқё нҢЁн„ҙмқҙлӢӨ.

Factory Methodк°Җ “к°қмІҙ н•ҳлӮҳлҘј м–ҙл–»кІҢ л§Ңл“Өм§Җ"м—җ м§‘мӨ‘н•ңлӢӨл©ҙ, Abstract FactoryлҠ” “м„ңлЎң м–ҙмҡёлҰ¬лҠ” к°қмІҙ м„ёнҠёлҘј м–ҙл–Ө кі„м—ҙлЎң л§Ңл“Өм§Җ"м—җ м§‘мӨ‘н•ңлӢӨ.

мҳҲлҘј л“Өм–ҙ GUIлҘј л§Ңл“ лӢӨкі  н•ҙліҙмһҗ.

  • Windows нҷҳкІҪм—җм„ңлҠ” WindowsButton, WindowsCheckbox, WindowsTextbox
  • Mac нҷҳкІҪм—җм„ңлҠ” MacButton, MacCheckbox, MacTextbox

мқҙл ҮкІҢ лІ„нҠј, мІҙнҒ¬л°•мҠӨ, н…ҚмҠӨнҠёл°•мҠӨк°Җ к°Ғк°Ғ л”°лЎң мЎҙмһ¬н•ҳм§Җл§Ң, мӢӨм ңлЎңлҠ” к°ҷмқҖ мҡҙмҳҒмІҙм ң мҠӨнғҖмқјлҒјлҰ¬ н•Ёк»ҳ мғқм„ұлҗҳм–ҙм•ј н•ңлӢӨ.

мқҙл•Ң WindowsButtonкіј MacCheckboxк°Җ м„һм—¬лІ„лҰ¬л©ҙ UI мҠӨнғҖмқјмқҙ к№Ём§„лӢӨ. Abstract FactoryлҠ” мқҙлҹ° л¬ём ңлҘј л§үкі , кҙҖл Ё к°қмІҙл“Өмқ„ н•ҳлӮҳмқҳ м ңн’Ҳкө°мңјлЎң мқјкҙҖлҗҳкІҢ мғқм„ұн•ҳкІҢ н•ҙмӨҖлӢӨ.
