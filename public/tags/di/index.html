мқҳмЎҙм„ұмқҙлһҖ?

мқҳмЎҙм„ұ = “мқҙкІҢ м—Ҷмңјл©ҙ лӮҳлҸ„ лӘ» лҸҢм•„к°ҖлҠ” кІғ”
мқҳмЎҙм„ұ кҙҖлҰ¬ = “к·ё ‘м—Ҷмңјл©ҙ м•Ҳ лҗҳлҠ” кІғ’мқҙ л°”лҖ” л•Ң лӮҳн•ңн…Ң м–јл§ҲлӮҳ нҚјм§ҖлғҗлҘј м ңм–ҙн•ҳлҠ” кІғ”
-> мқҳмЎҙ лҢҖмғҒмқҙ л°”лҖ” л•Ң м–јл§ҲлӮҳ л§ҺмқҖ кіімқ„ мҲҳм •н•ҙм•ј н•ҳлғҗк°Җ м„Өкі„ н’Ҳм§Ҳмқҳ мІҷлҸ„

Dependency Injection Pattern (мқҳмЎҙм„ұ мЈјмһ… нҢЁн„ҙ)

Dependency Injection(мқҳмЎҙм„ұ мЈјмһ…, DI)мқҖ к°қмІҙк°Җ мһҗмӢ мқҙ н•„мҡ”н•ң мқҳмЎҙм„ұмқ„ м§Ғм ‘ мғқм„ұн•ҳм§Җ м•Ҡкі , мҷёл¶Җм—җм„ң мЈјмһ…(inject)л°ӣлҠ” л””мһҗмқё нҢЁн„ҙмқҙлӢӨ. “мқҳмЎҙм„ұмқ„ мҠӨмҠӨлЎң л§Ңл“Өм§Җ л§җкі , лҲ„кө°к°Җк°Җ к°Җм ёлӢӨмӨ„ л•Ңк№Ңм§Җ кё°лӢӨл Өлқј.”

мҳҲлҘј л“Өм–ҙ:

# DI м—Ҷмқҙ: мқҳмЎҙм„ұмқ„ мҠӨмҠӨлЎң л§Ңл“ лӢӨ
class OrderService:
    def __init__(self):
        self.db = MySQLDatabase()   # м§Ғм ‘ мғқм„ұ вҶҗ л¬ём ң

# DI м Ғмҡ©: мқҳмЎҙм„ұмқ„ мЈјмһ…л°ӣлҠ”лӢӨ
class OrderService:
    def __init__(self, db):         # мҷёл¶Җм—җм„ң л°ӣлҠ”лӢӨ вҶҗ н•ҙкІ°
        self.db = db

# нҳём¶ң мёЎм—җм„ң мқҳмЎҙм„ұмқ„ кІ°м •н•ҳкі  мЈјмһ…
db = MySQLDatabase()
service = OrderService(db)

вҶ’ OrderServiceлҠ” MySQLDatabaseк°Җ лӯ”м§Җ м•Ң н•„мҡ”к°Җ м—ҶлӢӨ. мқён„°нҺҳмқҙмҠӨл§Ң м•Ңл©ҙ лҗңлӢӨ.
