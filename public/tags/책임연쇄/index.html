Chain of Responsibility Pattern (мұ…мһ„ м—°мҮ„ нҢЁн„ҙ)

Chain of Responsibility Pattern(мұ…мһ„ м—°мҮ„ нҢЁн„ҙ, COR)мқҖ мҡ”мІӯмқ„ мІҳлҰ¬н•  мҲҳ мһҲлҠ” к°қмІҙл“Өмқ„ мӮ¬мҠ¬(chain)мІҳлҹј м—°кІ°н•ҳкі , к°Ғ к°қмІҙк°Җ мҡ”мІӯмқ„ мІҳлҰ¬н•ҳм§Җ лӘ»н•ҳл©ҙ лӢӨмқҢ к°қмІҙлЎң мұ…мһ„мқ„ м „к°Җн•ҳлҠ” н–үлҸҷ л””мһҗмқё нҢЁн„ҙмқҙлӢӨ.

мҡ”мІӯмқ„ ліҙлӮҙлҠ” мӘҪ(sender)кіј мІҳлҰ¬н•ҳлҠ” мӘҪ(receiver)мқ„ лҠҗмҠЁн•ҳкІҢ кІ°н•©мӢңмјң, нҒҙлқјмқҙм–ёнҠёк°Җ м–ҙл–Ө к°қмІҙк°Җ мҡ”мІӯмқ„ мІҳлҰ¬н• м§Җ м§Ғм ‘ м•Ң н•„мҡ”к°Җ м—ҶкІҢ л§Ңл“ лӢӨ. нҠ№нһҲ мӨ‘мІ©лҗң if-else мЎ°кұҙл¬ёмқ„ мөңм Ғнҷ”н•ҳлҠ” лҚ° мӢӨл¬ҙм—җм„ң мһҗмЈј м“°мқёлӢӨ.

мҳҲлҘј л“Өм–ҙ:

class Handler:
    def __init__(self):
        self.next_handler = None
    def set_next(self, handler):
        self.next_handler = handler
        return handler  # л©”м„ңл“ң мІҙмқҙлӢқмқ„ мң„н•ҙ мқёмһҗлҘј к·ёлҢҖлЎң л°ҳнҷҳ
    def handle(self, request):
        # мІҳлҰ¬ лӘ»н•ҳл©ҙ лӢӨмқҢ н•ёл“Өлҹ¬лЎң мұ…мһ„ м „к°Җ
        if self.next_handler:
            return self.next_handler.handle(request)
        return None

class ConcreteHandlerA(Handler):
    def handle(self, request):
        if request == "A":
            return "AлҘј ConcreteHandlerAк°Җ мІҳлҰ¬"
        return super().handle(request)  # лӢӨмқҢ н•ёл“Өлҹ¬лЎң м „к°Җ

class ConcreteHandlerB(Handler):
    def handle(self, request):
        if request == "B":
            return "BлҘј ConcreteHandlerBк°Җ мІҳлҰ¬"
        return super().handle(request)

# нҒҙлқјмқҙм–ёнҠёлҠ” м–ҙлҠҗ н•ёл“Өлҹ¬к°Җ мІҳлҰ¬н•ҳлҠ”м§Җ м•Ң н•„мҡ”к°Җ м—ҶмқҢ
a = ConcreteHandlerA()
b = ConcreteHandlerB()
a.set_next(b)  # a вҶ’ b мІҙмқё кө¬м„ұ

print(a.handle("A"))  # AлҘј ConcreteHandlerAк°Җ мІҳлҰ¬
print(a.handle("B"))  # BлҘј ConcreteHandlerBк°Җ мІҳлҰ¬
print(a.handle("C"))  # None (мІҙмқё лҒқк№Ңм§Җ мІҳлҰ¬ м•Ҳ лҗЁ)

вҶ’ к°Ғ н•ёл“Өлҹ¬лҠ” мһҗмӢ мқҙ мІҳлҰ¬н•  мҲҳ мһҲлҠ”м§Җл§Ң нҢҗлӢЁн•ҳкі , лӘ»н•ҳл©ҙ super().handle()мқ„ нҶөн•ҙ лӢӨмқҢ н•ёл“Өлҹ¬лЎң л– л„ҳкёҙлӢӨ. нҒҙлқјмқҙм–ёнҠёлҠ” мІҙмқёмқҳ мӢңмһ‘м җм—җл§Ң мҡ”мІӯмқ„ ліҙлӮёлӢӨ.
