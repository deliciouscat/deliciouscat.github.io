Pydantic-AIм—җм„ң лӢӨмӨ‘ Agent нҳ‘м—…мқҳ к°ҖмһҘ кё°ліём Ғмқё нҢЁн„ҙмқҖ лҸ„кө¬ н•ЁмҲҳ лӮҙм—җм„ң лӢӨлҘё AgentлҘј нҳём¶ңн•ҳлҠ” “мң„мһ„(Delegation)“мқҙлӢӨ.

м„ёл¶Җ лӘ©м°Ё

1. Agent мң„мһ„мқҙлһҖ?

  • н•ҳлӮҳмқҳ Agentк°Җ нҠ№м • мһ‘м—…мқ„ лӢӨлҘё Agentм—җкІҢ л„ҳкё°лҠ” нҢЁн„ҙ
  • LangGraphмқҳ Agent м „нҷҳкіјмқҳ л№„көҗ
  • мң„мһ„мқҳ мһҘм җ: кҙҖмӢ¬мӮ¬ л¶„лҰ¬, м „л¬ёнҷ”

2. лҸ„кө¬ лӮҙм—җм„ң Agent нҳём¶ңн•ҳкё°

  • @agent.tool н•ЁмҲҳ м•Ҳм—җм„ң other_agent.run() нҳём¶ң
  • л¶ҖлӘЁ Agent вҶ” мһҗмӢқ Agent к°„ лҚ°мқҙн„° м „лӢ¬
  • мһҗмӢқ Agentмқҳ кІ°кіјлҘј лҸ„кө¬ л°ҳнҷҳк°’мңјлЎң ліҖнҷҳ

3. мң„мһ„ кө¬мЎ° м„Өкі„

  • мҳӨмјҖмҠӨнҠёл Ҳмқҙн„° Agent + м „л¬ё Agent нҢЁн„ҙ
  • л¶„лҘҳ(Router) Agent вҶ’ мІҳлҰ¬(Worker) Agent нҢЁн„ҙ
  • кі„мёөм Ғ мң„мһ„ (Manager вҶ’ Team Lead вҶ’ Worker)

4. м»Ён…ҚмҠӨнҠё м „нҢҢ

  • л¶ҖлӘЁмқҳ RunContextлҘј мһҗмӢқ Agentм—җ м „лӢ¬н•ҳлҠ” л°©лІ•
  • кіөнҶө мқҳмЎҙм„ұ кіөмң  vs кІ©лҰ¬
  • мһҗмӢқ Agentмқҳ л©”мӢңм§Җ мқҙл Ҙ кҙҖлҰ¬

5. м—җлҹ¬ м „нҢҢмҷҖ мһ¬мӢңлҸ„

  • мһҗмӢқ Agent мӢӨн–ү мӢӨнҢЁ мӢң л¶ҖлӘЁ Agentмқҳ лҢҖмқ‘
  • мң„мһ„ мӢӨнҢЁм—җ лҢҖн•ң Fallback м „лһө
  • мһ¬мӢңлҸ„ мҳҲмӮ°мқҳ л¶„л°°

6. мӢӨм „ мҳҲм ң

  • кі к°қ м„ңл№„мҠӨ: л¶„лҘҳ Agent вҶ’ (нҷҳл¶Ҳ Agent / кё°мҲ м§Җмӣҗ Agent / мқјл°ҳл¬ёмқҳ Agent)
  • л¬ём„ң мІҳлҰ¬: нҢҢмӢұ Agent вҶ’ л¶„м„қ Agent вҶ’ ліҙкі м„ң Agent
