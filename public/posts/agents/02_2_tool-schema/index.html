к°ңмҡ”

Pydantic-AIлҠ” н•ЁмҲҳ мӢңк·ёлӢҲмІҳмҷҖ docstringмқ„ кё°л°ҳмңјлЎң лҸ„кө¬мқҳ JSON мҠӨнӮӨл§ҲлҘј мһҗлҸҷ мғқм„ұн•ңлӢӨ. мқҙ мҠӨнӮӨл§Ҳк°Җ LLMмқҙ лҸ„кө¬лҘј мқҙн•ҙн•ҳкі  мҳ¬л°”лҘҙкІҢ нҳём¶ңн•ҳлҠ” “мӮ¬мҡ©м„ӨлӘ…м„ң” м—ӯн• мқ„ н•ңлӢӨ.

м„ёл¶Җ лӘ©м°Ё

1. мҠӨнӮӨл§Ҳ мһҗлҸҷ мғқм„ұ мӣҗлҰ¬

  • н•ЁмҲҳ мӢңк·ёлӢҲмІҳ вҶ’ JSON Schema ліҖнҷҳ кіјм •
  • Annotated нғҖмһ… нһҢнҠёмқҳ нҷңмҡ©
  • Pydantic-AIк°Җ мқҪлҠ” м •ліҙ: нҢҢлқјлҜён„°лӘ…, нғҖмһ…, кё°ліёк°’, docstring

2. docstringмқҳ м—ӯн• 

  • Google мҠӨнғҖмқј docstring мһ‘м„ұлІ•
  • Args: м„№м…ҳмқҙ нҢҢлқјлҜён„° м„ӨлӘ…мңјлЎң ліҖнҷҳлҗҳлҠ” кіјм •
  • н•ЁмҲҳ м„ӨлӘ…(мІ« мӨ„)мқҙ лҸ„кө¬ м„ӨлӘ…мқҙ лҗҳлҠ” к·ңм№ҷ

3. нғҖмһ… нһҢнҠёлҘј нҶөн•ң мҠӨнӮӨл§Ҳ м ңм–ҙ

  • Literal вҖ” м„ нғқм§Җ м ңн•ң
  • Optional / Union вҖ” м„ нғқм Ғ нҢҢлқјлҜён„°
  • Annotated[int, Field(ge=0, le=100)] вҖ” к°’ лІ”мң„ м ңм•Ҫ

4. @agent.tool vs @agent.tool_plain

  • tool: RunContextлҘј мІ« лІҲм§ё мқёмһҗлЎң л°ӣмқҢ
  • tool_plain: м»Ён…ҚмҠӨнҠё м—Ҷмқҙ мҲңмҲҳ н•ЁмҲҳлЎң лҸҷмһ‘
  • м–ём ң м–ҙл–Ө кІғмқ„ м„ нғқн•ҙм•ј н•ҳлҠ”к°Җ

5. м»ӨмҠӨн…Җ мҠӨнӮӨл§Ҳ мҳӨлІ„лқјмқҙл“ң

  • Tool() к°қмІҙлҘј м§Ғм ‘ мғқм„ұн•ҳм—¬ м„ёл°Җн•ң м ңм–ҙ
  • нҢҢлқјлҜён„° мқҙлҰ„ ліҖкІҪ, м„ӨлӘ… м»ӨмҠӨн…Җ

6. мўӢмқҖ лҸ„кө¬ мҠӨнӮӨл§Ҳ м„Өкі„ мӣҗм№ҷ

  • LLMмқҙ мқҙн•ҙн•ҳкё° мү¬мҡҙ нҢҢлқјлҜён„° л„Өмқҙл°Қ
  • лӘ…нҷ•н•ң л°ҳнҷҳк°’ м„ӨлӘ…мқҳ мӨ‘мҡ”м„ұ
  • кіјлҸ„н•ң нҢҢлқјлҜён„° vs м Ғм Ҳн•ң м¶”мғҒнҷ”
