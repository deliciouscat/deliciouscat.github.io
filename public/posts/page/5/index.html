Observer Pattern (кҙҖм°°мһҗ нҢЁн„ҙ)

Observer Pattern(кҙҖм°°мһҗ нҢЁн„ҙ)мқҖ к°қмІҙ к°„мқҳ мқјлҢҖлӢӨ(one-to-many) мқҳмЎҙ кҙҖкі„лҘј м •мқҳн•ҳм—¬, н•ң к°қмІҙмқҳ мғҒнғңк°Җ ліҖкІҪлҗҳл©ҙ к·ём—җ мқҳмЎҙн•ҳлҠ” лӘЁл“  к°қмІҙл“Өм—җкІҢ мһҗлҸҷмңјлЎң м•ҢлҰјмқ„ м „лӢ¬н•ҳкі  м—…лҚ°мқҙнҠён•ҳлҠ” н–үлҸҷ л””мһҗмқё нҢЁн„ҙмқҙлӢӨ.

нҒҙлқјмқҙм–ёнҠёк°Җ Subject(кҙҖм°° лҢҖмғҒ)кіј Observer(кҙҖм°°мһҗ)лҘј лҠҗмҠЁн•ҳкІҢ кІ°н•©лҗң л°©мӢқмңјлЎң мғҒнҳёмһ‘мҡ©н•  мҲҳ мһҲкІҢ н•ңлӢӨ.

мҳҲлҘј л“Өм–ҙ:

class Subject:
    def __init__(self):
        self.observers = []
    def attach(self, observer): self.observers.append(observer)
    def notify(self): 
        for observer in self.observers:
            observer.update(self)

class Observer:
    def update(self, subject): pass

class ConcreteSubject(Subject):
    def __init__(self): 
        super().__init__()
        self.state = 0
    def set_state(self, state): 
        self.state = state
        self.notify()  # мғҒнғң ліҖкІҪ мӢң мһҗлҸҷ м•ҢлҰј

class ConcreteObserver(Observer):
    def __init__(self, name): self.name = name
    def update(self, subject): 
        print(f"{self.name}: мғҒнғңк°Җ {subject.state}лЎң ліҖкІҪлҗЁ")

# нҒҙлқјмқҙм–ёнҠёлҠ” SubjectмҷҖ ObserverлҘј кө¬л¶„н•ҳм§Җ м•ҠмқҢ
stock = ConcreteSubject()
investor1 = ConcreteObserver("нҲ¬мһҗмһҗA")
investor2 = ConcreteObserver("нҲ¬мһҗмһҗB")

stock.attach(investor1)  # кҙҖм°°мһҗ л“ұлЎқ(нҲ¬мһҗмһҗAк°Җ мЈјмӢқмқ„ кҙҖм°°)
stock.attach(investor2)  # кҙҖм°°мһҗ л“ұлЎқ
stock.set_state(100)     # нҲ¬мһҗмһҗA: мғҒнғңк°Җ 100мңјлЎң ліҖкІҪлҗЁ
                         # нҲ¬мһҗмһҗB: мғҒнғңк°Җ 100мңјлЎң ліҖкІҪлҗЁ

вҶ’ ConcreteSubjectмҷҖ ConcreteObserverлҠ” лӘЁл‘җ к°Ғк°Ғ Subject, Observer мқён„°нҺҳмқҙмҠӨлҘј кө¬нҳ„н•ҙм„ң л°ңн–ү-кө¬лҸ… л©”м»ӨлӢҲмҰҳмқ„ нҶөн•ҙ лҠҗмҠЁн•ҳкІҢ кІ°н•©лҗңлӢӨ.

Vibe Codingмқ„ мң„н•ң л””мһҗмқё нҢЁн„ҙ - ліөн•©мІҙ(Composite)

September 20, 2025
Computer Science

Composite Pattern(ліөн•©мІҙ нҢЁн„ҙ)мқҖ к°қмІҙл“Өмқ„ нҠёлҰ¬ кө¬мЎ°лЎң кө¬м„ұн•ҳм—¬ л¶Җл¶„кіј м „мІҙлЎң кі„мёөмқ„ н‘ңнҳ„н•ҳлҠ” л””мһҗмқё нҢЁн„ҙмқҙлӢӨ. нҒҙлқјмқҙм–ёнҠёк°Җ к°ңлі„ к°қмІҙмҷҖ ліөн•© к°қмІҙлҘј лҸҷмқјн•ң л°©мӢқмңјлЎң лӢӨлЈ° мҲҳ мһҲлӢӨ.

мҳҲлҘј л“Өм–ҙ:

class Component:
    def execute(self): pass

class Leaf(Component):
    def execute(self): return 1

class Composite(Component):
    def __init__(self): self.children = []
    def add(self, c): self.children.append(c)
    def execute(self): return sum(c.execute() for c in self.children)

# нҒҙлқјмқҙм–ёнҠёлҠ” LeafмҷҖ CompositeлҘј кө¬л¶„н•ҳм§Җ м•ҠмқҢ
items = [Leaf(), Composite(), Leaf()]  # лӘЁл‘җ Component
items[1].add(Leaf())    # Compositeм—җ Leaf м¶”к°Җ: children = [leaf1]
items[1].add(Leaf())    # Compositeм—җ Leaf м¶”к°Җ: children = [leaf1, leaf2]
total = sum(item.execute() for item in items)
print(total)  # 1 + 2 + 1 = 4

-> LeafмҷҖ CompositeлҠ” л‘ҳ лӢӨ Component(execute л©”м„ңл“ңлҘј к°Җм§„)лҘј мғҒмҶҚн•ҙм„ң execute л©”м„ңл“ңлҘј нҳём¶ңн•  мҲҳ мһҲкі , мң„кі„м—җ мғҒкҙҖм—Ҷмқҙ л©”м„ңл“ңлҘј нҳём¶ң к°ҖлҠҘн•ҳлӢӨ.
